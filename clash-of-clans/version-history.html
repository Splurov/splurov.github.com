Version History — Clash Calc

4.1
29 January, 2014

Game version 5.172 updates:

  • Added Minion level 6.
  • Added Freeze Spell level 5.
  • Increased Hog Rider training cost.

4.0
28 January, 2014

  • Increased calculator performance (faster loading, better responsiveness).
  • Animated saving and deleting of saved compositions.
  • Slightly redesigned Setting Mode.
  • Minor mobile-related improvements.
  • Simplified keyboard navigation.
  • Dropped Opera 12.1 and Safari 5.1 support (currently supported browsers is Safari 6+, Chrome 21+, Firefox 24+, IE 10+, Android 4+ and Opera 17+).

3.5
6 January, 2014

  • When possible, troops distributed between the barracks into equal parts.
  • Improved usability of plus/minus buttons (no more frozen state on desktop or failed attempts to scroll on mobile).

3.4
29 December, 2013

  • Added support for “gem-boosted” Spell Factory.
  • Improved support for non-Apple devices (Android and other mobile devices).

3.3
22 December, 2013

  • Added support for “gem-boosted” Barracks and Dark Barracks.

3.2.1
19 November, 2013

  • Fixed a bug with the inability to focus the input field in some cases.
  • Fixed a bug related to preset levels according to the Town Hall level 2.
  • Fixed a bug in Safari 5 and 6 related to setting mode.

3.2
16 November, 2013

  • Added Setting Mode with the possibility of preset levels of buildings, troops and spells according to the level of the Town Hall.
  • Values from the “Subtract” columns not preserved between browser sessions anymore.
  • Zeros no longer used for empty values in the text fields.
  • Added display of the queue length for barracks.
  • Dropped IE 9 support (currently supported browsers is Safari 5.1+, Chrome 21+, Firefox 17+, IE 10+, Android 4+ and Opera 12.1+).

6 November, 2013

  • The cost of spells now in Elixir instead of Gold.

  • Updated home screen icons for iOS 7.
  • Added tooltip for troops icons.

2.4
14 April, 2013

  • Added awesome feature to “subtract” troops, which you already have. For example, you want to get 100 Barbarians and 100 Archers. And you already have 20 Barbarians. In the column “Quantity”, as usual, make 100 Barbarians and 100 Archers. And in the column “Subtract” 20 Barbarians. Now the distribution between the barracks are considered to be only for 80 Barbarians and 100 Archers. This is very useful, when after the battle you have remained part of the army.
